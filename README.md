This repository contains MATLAB code and processed datasets used to analyze longitudinal turbulent heat flux in the atmospheric surface layer, including RANS analysis, spectral/cospectral analysis, and implementation of the cospectral budget (CSB) model described in the associated manuscript. The main script (`MAIN.m`) reproduces the primary analyses and figures by loading the provided datasets: `data_sonic_SLTEST.mat` (SLTEST sonic anemometer data), `data_sonic_grass_site.mat` (Duke Forest grass-site sonic data), and `Utah_data.mat` (hot-wire measurements from SLTEST). The code has been tested in MATLAB (R2023a or later) and requires standard toolboxes such as Signal Processing and Optimization.
